# NuGet Alternatives to SunamoDebugCollection

This document lists popular NuGet packages that provide similar functionality to SunamoDebugCollection.

## Overview

Collection debugging utilities

## Alternative Packages

### System.Diagnostics.Debug
- **NuGet**: System.Diagnostics.Debug
- **Purpose**: Built-in debugging
- **Key Features**: Assert, WriteLine, conditional compilation

### ObjectDumper
- **NuGet**: ObjectDumper.NET
- **Purpose**: Object visualization
- **Key Features**: Dump objects to string, property inspection

### Serilog
- **NuGet**: Serilog
- **Purpose**: Structured logging
- **Key Features**: Log complex objects, structured data

### BenchmarkDotNet
- **NuGet**: BenchmarkDotNet
- **Purpose**: Performance analysis
- **Key Features**: Collection performance benchmarking

## Comparison Notes

ObjectDumper for quick inspection. Serilog for production debugging.

## Choosing an Alternative

Consider these alternatives based on your specific needs:
- **System.Diagnostics.Debug**: Built-in debugging
- **ObjectDumper**: Object visualization
- **Serilog**: Structured logging
